# x团队多智能体协同提示词

该仓库提供一套完全中文的系统提示词，用于搭建由“分析与规划”“执行与处理”“反思”“记忆”四个Agent组成的 x 团队。团队的目标是在接收到用户需求后，高效完成需求分析、任务拆解、执行落地、结果复核以及经验沉淀。

## 场景设定
- x 团队以项目团队的方式运作，每个 Agent 拥有清晰的职责边界和沟通接口。
- 分析与规划 Agent 需要根据用户输入和历史记忆，产出带有任务类型与目标的任务列表。任务类型仅支持“流程发起”“流程审批”“数据分析与报告呈现”“业务分析与报告呈现”。
- 执行与处理 Agent 根据任务类型选择适配的岗位身份完成工作，并输出可验证的成果。
- 反思 Agent 负责校验执行结果与计划是否一致，并提出改进或复盘建议。
- 记忆 Agent 负责在整个流程中记录关键信息、经验与决策，供后续查询和复用。

## 目录结构
- `prompts/分析与规划智能体提示词.md`
- `prompts/执行与处理智能体提示词.md`
- `prompts/反思智能体提示词.md`
- `prompts/记忆智能体提示词.md`

## 使用指引
1. 在多智能体框架中，将对应文件的内容作为系统提示词注入相应 Agent，保持输出结构不变。
2. 若需扩展领域知识，可在不改变角色分工与任务类型约束的前提下追加补充说明。
3. 结合记忆 Agent 的积累以及反思 Agent 的反馈，持续迭代提示词外的业务知识，以增强团队整体表现。
